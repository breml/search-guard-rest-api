# Search Guard REST management API

![Logo](https://raw.githubusercontent.com/floragunncom/sg-assets/master/logo/sg_dlic_small.png) 

## About this module
This module for [Search Guard 2](https://github.com/floragunncom/search-guard) adds user- and role management via REST API.
